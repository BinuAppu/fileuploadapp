# fileuploadapp
First Projected Created using Googles Antigravity. This tool will help to upload and download files from machine using Flask App. 

### For ease add the following line into .bashrc to run the Flask App from current directory.
#### Considering the file is kept at /opt/tools folder.
```bash
alias appufileupload='/usr/bin/python3 /opt/tools/appu_file_app/app.py'
```
### Once the file is added, run the command to spin the web server.
